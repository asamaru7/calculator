속도 단위변환

단위변환 2018.10.31. 09:18 2018.12.01. 09:18 Asamaru
  • m/s (미터 매 초 : meter per second)
    미터 매 초 또는 초속 미터는 국제단위계의 속도 대한 SI 유도 단위로 1초 동안 운동한 미터 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • m/h (미터 매 시 : meter per hour)
    미터 매 시 또는 시속 미터는 국제단위계의 속도 대한 SI 유도 단위로 1시간 동안 운동한 미터 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • km/s (킬로미터 매 초 : kilometer per second)
    킬로미터 매 초 또는 초속 킬로미터는 1초 동안 운동한 킬로미터 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • km/h (킬로미터 매 시 : kilometer per hour)
    킬로미터 매 시 또는 시속 미터는 1시간 동안 운동한 킬로미터 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • in/s (인치 매 초 : inch per second)
    인치 매 초 또는 초속 인치는 1초 동안 운동한 인치 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • in/h (인치 매 시 : inch per hour)
    인치 매 시 또는 시속 인치는 1시간 동안 운동한 인치 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • ft/s (피트 매 초 : feet per second)
    피트 매 초 또는 초속 피트는 1초 동안 운동한 피트 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • ft/h (피트 매 시 : feet per hour)
    피트 매 시 또는 시속 피트는 1시간 동안 운동한 피트 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • mi/s (마일 매 초 : mile per second)
    마일 매 초 또는 초속 마일는 1초 동안 운동한 마일 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • mi/h (마일 매 시 : mile per hour)
    마일 매 시 또는 시속 마일는 1시간 동안 운동한 마일 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다.
  • kn (노트 : knot)
    노트는 1시간 동안 운동한 해리(1,852미터) 수로 표현하는 속력(스칼라)과 속도(벡터)의 측정 단위이다. 해양 또는 항공 분야에서 주로 사용되는 속도의 단위지만 국제단위계에는 속하지 않는다.
  • mach (마하 : mach)
    음속에 대한 운동 물체의 속도의 비로 나타내는 속도 단위로 주로 비행기, 로켓, 고속 기류 따위의 속도를 잴 때 사용된다. 1마하는 약 340m/s이고 시속 1,224km/h에 해당한다.
