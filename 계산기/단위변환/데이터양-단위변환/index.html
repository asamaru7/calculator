데이터양 단위변환

단위변환 2018.10.31. 11:18 2018.12.01. 11:18 Asamaru
  • bit (비트 : bit)
    binary digit의 약칭으로 2진수에서의 숫자 0, 1과 같이 신호를 나타내는 최소 단위다.
  • B (바이트 : byte)

    8비트(bit)로 구성되는 정보 단위로 대개 1옥텟인 8비트가 1바이트이다. 4비트는 니블(nibble), 2바이트는 일부 전자통신기기에서 워드(word)를 가리킨다.

    바이트는 컴퓨터의 기억장치의 크기를 나타내는 단위로 자주 쓰이며, 실질적 의미는 ASCII 문자 하나를 나타낼 수 있다는 것이다. 따라서 여러 바이트를 한 워드로 사용하고 있는 현재에도 대부분의 컴퓨터 하드웨에서 메모리의 주소 단위로 사용된다.

  • KB (킬로바이트 : kilobyte)
    킬로바이트는 1,000바이트를 뜻하는 정보의 단위이다.
  • KiB (키비바이트 : kibibyte)
    키비바이트는 1,024바이트를 뜻하는 정보의 단위이다.
  • MB (메가바이트 : megabyte)
    메가바이트는 1,000킬로바이트, 106(1,000,000)바이트를 뜻하는 정보의 단위이다.
  • MiB (메비바이트 : mebibyte)
    메비바이트는 1,024키비바이트, 220(10242 = 1,048,576)바이트를 뜻하는 정보의 단위이다.
  • GB (기가바이트 : gigabyte)
    기가바이트는 1,000메가바이트, 109(1,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • GiB (기비바이트 : gibibyte)
    기비바이트는 1,024메비바이트, 230(10243 = 1,073,741,824)바이트를 뜻하는 정보의 단위이다.
  • TB (테라바이트 : terabyte)
    테라바이트는 1,000기가바이트, 1012(1,000,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • TiB (테비바이트 : tebibyte)
    테비바이트는 1,024기비바이트, 240(10244 = 1,099,511,627,776)바이트를 뜻하는 정보의 단위이다.
  • PB (페타바이트 : petabyte)
    페타바이트는 1,000테라바이트, 1015(1,000,000,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • PiB (페비바이트 : pebibyte)
    페비바이트는 1,024테비바이트, 250(10245 = 1,125,899,906,842,624)바이트를 뜻하는 정보의 단위이다.
  • EB (엑사바이트 : exabyte)
    엑사바이트는 1,000페타바이트, 1018(1,000,000,000,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • EiB (엑비바이트 : exbibyte)
    엑비바이트는 1,024페비바이트, 260(10246 = 1,152,921,504,606,846,976)바이트를 뜻하는 정보의 단위이다.
  • ZB (제타바이트 : zettabyte)
    제타바이트는 1,000엑사바이트, 1021(1,000,000,000,000,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • ZiB (제비바이트 : zebibyte)
    제비바이트는 1,024엑비바이트, 270(10247 = 1,180,591,620,717,411,303,424)바이트를 뜻하는 정보의 단위이다.
  • YB (요타바이트 : yottabyte)
    요타바이트는 1,000제타바이트, 1024(1,000,000,000,000,000,000,000,000)바이트를 뜻하는 정보의 단위이다.
  • YiB (요비바이트 : yobibyte)
    요비바이트는 1,024제비바이트, 280(10248 = 1,208,925,819,614,629,174,706,176)바이트를 뜻하는 정보의 단위이다.
